# Simple Chatbot Flutter Client

This is a Flutter client for the Simple Chatbot Genkit sample.

## Getting Started

1.  **Install Flutter:** Make sure you have the Flutter SDK installed. See the [Flutter documentation](https://flutter.dev/docs/get-started/install) for instructions.

2.  **Install Dependencies:** Navigate to this directory and run:

    ```sh
    flutter pub get
    ```

3.  **Run the Genkit Backend:** Make sure the Genkit backend is running. See the main `README.md` in the `simple-chatbot` directory for instructions.

4.  **Run the App:**

    ```sh
    flutter run
